# Exercício 4 — Vetor sem duplicados

## 📌 Descrição
O programa solicita 10 números ao usuário, armazena em um vetor e cria um novo vetor removendo os valores repetidos.

---

## 🛠️ Tecnologias Utilizadas

- JavaScript
- HTML
- Visual Studio Code
- Navegador

---

## 💡 Conceitos Utilizados

- Vetores
- Estrutura de repetição `for`
- Estrutura de repetição aninhada
- Comparação de valores

---

## ▶️ Como Executar

1. Abrir o arquivo `index.html`
2. Executar no navegador
3. Digitar os números solicitados
4. Abrir o console com `F12`

---

## ✅ Resultado Esperado

Exibir:
- vetor original
- vetor sem números repetidos
