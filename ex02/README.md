# Exercício 2 — Vetor Numérico

## 📌 Descrição
O programa cria um vetor com 7 números e exibe todos os valores armazenados.

---

## 🛠️ Tecnologias Utilizadas

- JavaScript
- HTML
- Visual Studio Code
- Navegador

---

## 💡 Conceitos Utilizados

- Vetores
- Estrutura de repetição `for`
- Exibição no console

---

## ▶️ Como Executar

1. Abrir o arquivo `index.html`
2. Executar no navegador
3. Abrir o console com `F12`

---

## ✅ Resultado Esperado

Exibir todos os valores armazenados no vetor.
