# Exercício 1 — Soma dos números pares

## 📌 Descrição
O programa solicita 10 números ao usuário, armazena em um vetor e calcula a soma apenas dos números pares.

---

## 🛠️ Tecnologias Utilizadas

- JavaScript
- HTML
- Visual Studio Code
- Navegador

---

## 💡 Conceitos Utilizados

- Vetores
- Estrutura de repetição `for`
- Condição `if`
- Operador `%`

---

## ▶️ Como Executar

1. Abrir o arquivo `index.html`
2. Executar no navegador
3. Digitar os números solicitados
4. Abrir o console com `F12`

---

## ✅ Resultado Esperado

Exibir:
- os números armazenados no vetor
- a soma dos números pares
