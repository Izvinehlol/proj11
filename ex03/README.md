# Exercício 3 — Verificar quantidade de números

## 📌 Descrição
O programa solicita 10 números ao usuário, armazena em um vetor e verifica quantas vezes um número informado aparece.

---

## 🛠️ Tecnologias Utilizadas

- JavaScript
- HTML
- Visual Studio Code
- Navegador

---

## 💡 Conceitos Utilizados

- Vetores
- Estrutura de repetição `for`
- Comparação de valores
- Contador

---

## ▶️ Como Executar

1. Abrir o arquivo `index.html`
2. Executar no navegador
3. Digitar os números solicitados
4. Abrir o console com `F12`

---

## ✅ Resultado Esperado

Exibir:
- os números do vetor
- quantas vezes o número pesquisado aparece
